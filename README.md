## Installation
1. Clone the repo: 
   - https: `git clone https://github.com/OMS-NetZero/FAIR.git`.
   - ssh: `git clone git@github.com:OMS-NetZero/FAIR.git`

1. `cd` into the directory.

1. A typical `python setup.py install` should suffice.
